Problem 1 - Format with placeholders

  • Write a function that formats a string. The function should have placeholders, as shown in the example
  • Add the function to the String.prototype
Example 1 Input: var options = {name: 'John'}; 'Hello, there! Are you #{name}?'.format(options); Output: 'Hello, there! Are you John'
Example 2 Input: var options = {name: 'John', age: 13}; 'My name is #{name} and I am #{age}-years-old'.format(options); Output: 'My name is John and I am 13-years-old'
